# Admin Dashboard

O resultado final do projeto é uma dashboard de administração desenvolvida com o objetivo de fornecer uma interface intuitiva e responsiva para gerenciar dados e visualizar informações relevantes. Esta aplicação front-end foi projetada para integrar-se a um sistema de backend e utiliza as seguintes tecnologias:

Antes de mais tem que criar um usuario

## Tecnologias Utilizadas

- **Next.js**: Um framework React para desenvolvimento de aplicações web com renderização do lado do servidor (SSR) e geração de sites estáticos.
- **Tailwind CSS**: Uma biblioteca de CSS utilitário que permite a criação de interfaces personalizadas de forma rápida e eficiente.
- **Shadcn UI**: Um conjunto de componentes de interface do usuário que fornece uma base para construir uma experiência de usuário coesa e acessível.

## Funcionalidades

- **Autenticação Segura**: Implementação de middleware de autenticação para proteger rotas e garantir que apenas usuários autorizados tenham acesso à dashboard.
- **Interface Responsiva**: Design adaptável que proporciona uma experiência de usuário otimizada em dispositivos móveis e desktops.
- **Visualização de Dados**: Componentes que exibem dados relevantes, como receita total, assinaturas e vendas, em cartões informativos e gráficos.
